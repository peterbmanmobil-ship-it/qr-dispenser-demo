# QR Dispenser Demo – Branded + Dropdown

This package contains a ready-to-drop-in set of files for GitHub Pages.

## What’s included
- White, Tork-inspired styling
- Centered small logo in header
- Country selector as a dropdown (alphabetical)
- Placeholder: “Choose country” (Open button disabled until selected)

## URLs
- Example: `index.html?p=564500`
- Example: `index.html?p=581500`

## Install
Copy all files into your GitHub repo folder (same folder as `index.html`), replacing existing files.
Then commit + push.
